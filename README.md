## Installation

#### Manual run install Golang:

    1. clone repository  https://github.com/afitra/sv_backend_test.git
    2. masuk ke folder hasil clone
    3. setup .env file sesuai di file .env.example  
    4. go mod download
    5. go run main.go
    6. enjoy with golang API
